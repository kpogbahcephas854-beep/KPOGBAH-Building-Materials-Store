## Challenges
- Setting up footer menus with correct links required creating separate navigation menus for FAQs, Help Center, and Shopping Guide
- Removing "Sold Out" labels required disabling inventory tracking on all products
- Customizing the homepage hero section with background image and welcome text
- Fixing invalid URI errors when adding menu links with spaces in page names

---

## Lessons Learned
- Shopify is a powerful no-code platform that allows full e-commerce store setup without writing code
- Navigation menus and footer menus are managed separately in Shopify Admin
- Product inventory settings affect how products appear to customers (Sold Out vs. available)
- Theme customization through the Online Store Editor allows visual changes without coding
- Good product descriptions and images are essential for a professional-looking store
